A program using NASM that will do a selection sort of a sequence of lowercase characters. program should

1. Prompt the user to enter a sequence of characters.

2. Any non-lower case alphabetic characters should be removed before sorting.

3. Display to the user the pre-processed string (with only lower-case characters) that you will sort.

4. Perform a selection sort of the string and display the sequence every time a swap is made between two characters in different positions.

5. You must use the selection sort technique (i.e. you cannot use bubble sort, insertion sort, etc.)

6. It should be able to sort up to 500 characters (i.e. don’t come up with a solution that just works with a few characters).

7. The program should loop three times allowing the user to sort 3 different strings
